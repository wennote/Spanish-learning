Read Me

This website is a personal Spanish study dashboard based on your first five days of notes. It is designed to keep your early learning material in one place so you can review verbs, vocabulary, sentence patterns, question forms, short reading passages, pronunciation, and writing practice without searching through separate files. The site is now organized as multiple pages instead of a single long file, so each section is easier to review and edit.

The overall workflow is simple. Start with Read Me for orientation, move to Vocabulary for word review, use Grammar for reference and verb tables, go to Pronunciation to listen and record, use Review and Flashcards to focus selected items, practice them in Stories, Practice, and Writing, check Summary for learning suggestions, and use Daily Notes as lesson reference.

Current Sections

  • Vocabulary: words, verbs, and short example phrases from Days 1 to 5.
  • Grammar: quick explanations for verb endings, ser/estar, questions, and past tense.
  • Pronunciation: key words and model sentences with audio playback.
  • Writing: short self-introduction and daily routine paragraph practice.
  • Summary: learning results, quiz history, and next-step suggestions.
  • Daily Notes: lesson-by-lesson summaries from your study files.

How To Use

  • Use topic filters on Vocabulary to focus on verbs, feelings, places, or questions.
  • Use the speaker icons to hear both the target word and the example sentence.
  • Use Grammar as a reference page when reading or writing, especially for verb endings.
  • Use Pronunciation recording to compare your own voice with the model playback.
  • Use Writing after Practice so you can apply the same grammar and vocabulary actively.

Selection-Based Study Block

The pages below all depend on the checkboxes you mark in Vocabulary, Grammar, and Pronunciation. They work as one study pipeline rather than isolated pages.

  • Review: gathers the selected items into one focused explanation page.
  • Flashcards: uses selected vocabulary first, so your card review matches what you picked.
  • Stories: keeps full lesson context available, but highlights the selected items as the main reading focus.
  • Practice: builds customized exercises and quiz questions from the same selected items.
  • Summary: reports selected counts, quiz results, and next-step suggestions based on that workflow.

What Has Been Customized

  • Vocabulary now includes audio playback, alphabetical sorting, one example sentence audio, and topic filters.
  • Grammar was redesigned to use the page width better and now includes reference tables.

  • Writing includes prompt rotation, model answers, and quick grammar suggestions.

Current Limits

  • Audio uses the browser speech engine, so voice quality depends on the browser and system.
  • Writing suggestions are rule-based and helpful for common beginner mistakes, but not full grammar correction.
  • Recorded pronunciation is stored only in the current browser session unless exported manually.
  • The site is static HTML, CSS, and JS, so there is no server-side storage or user account system.
  • Vocabulary and grammar content are still focused on the first five study days, not a full course.
